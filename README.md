# Examples Node Stream API

## Description

This repo illustrates nodejs stream api mechanics.

## Start

### Development

1. Preparing

```bash
$ make prepare
```

2. Start application in development mode

```bash
$ make start
```

### Docker

1. Preparing

```bash
$ make prepare
```

2. Build images

```bash
$ make build
```

3. Start containers

```bash
$ make up
```
